# EasyResume - Resume Builder

A full-stack resume builder built using React, TypeScript, Node.js, Express, and MongoDB.

## Features

- User Authentication (JWT)
- Register / Login System
- Protected Routes
- Resume Builder Form
- Live Resume Preview
- Resume Save & Update
- MongoDB Database Integration
- Responsive UI
- Resume Templates
- PDF Export

---

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
